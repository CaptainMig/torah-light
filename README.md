# Torah Light ✡

A living, interactive Jewish curriculum built for Ariah & Isabelle.

## Features
- **Aleph-Bet** — Interactive Hebrew alphabet with meanings and gematria
- **Torah** — All 54 weekly portions with kid-friendly summaries
- **Live Torah Reading** — Real-time text from Sefaria API
- **Prayer** — 10 essential prayers with Hebrew, transliteration, and meaning
- **Globe** — 3D interactive globe of Jewish historical sites
- **Russian Heritage** — Timeline connecting maternal roots to Israel
- **Holidays** — Full Jewish calendar with live Shabbat times via Hebcal API
- **Values** — 12 core Jewish values with practical applications
- **Heart** — Emotional wisdom from Torah/Talmud for real feelings
- **Weekly Lesson** — Rotating teachings on Jewish concepts
- **Shabbat** — Guide to the weekly day of rest

## APIs
- [Sefaria](https://www.sefaria.org) — Live Torah text, parasha, haftarah, daf yomi
- [Hebcal](https://www.hebcal.com) — Shabbat times, Hebrew date, candle lighting

## Deploy
Connected to Vercel for automatic deployment on push.

## Stack
React 18 · Single-page app · No external UI libraries
